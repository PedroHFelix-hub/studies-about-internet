# 🌐 Studies About Internet

> A front-end study project exploring the history, evolution, and impact of the internet — built with pure HTML, CSS, and JavaScript.

---

## 📖 About the Project

**Studies About Internet** is a personal academic project created to deepen my understanding of how the internet works, where it came from, and how it shapes the modern world. Each section of the site covers a specific topic, organized into dedicated pages with structured content.

This project also serves as a practical exercise to improve my front-end development skills — from semantic HTML structure to responsive CSS layouts and basic JavaScript interactions.

---

## 📄 Pages

| Page | Description |
|------|-------------|
| `index.html` | Landing page — project overview |
| `internetemerge.html` | How the internet emerged — ARPANET, TCP/IP, Cold War context |
| `webevolution.html` | Web evolution — Web 1.0, 2.0, 3.0 and key technologies |
| `computer and data.html` | Computers and data — components, binary, Big Data |
| `developerroles.html` | Developer roles — Front-end, Back-end, Full-stack and more |

---

## 🛠️ Technologies Used

- **HTML5** — Semantic page structure
- **CSS3** — Styling, Flexbox layout, gradients, textures, and animations
- **JavaScript** — Tooltips, back-to-top button, and active page highlighting
- **CSS Reset** — Custom reset for consistent cross-browser rendering

---

## ✨ Features

- Responsive header with logo and social media links
- Navigation bar with smooth hover effects and tooltips
- Animated page entry (`fadeInUp`)
- CSS texture on the page background (circuit/dot pattern)
- SVG noise texture on the header for a tactile feel
- Back-to-top button (appears on scroll)
- Active page button highlighting via JavaScript

## 🚀 How to Run

No dependencies or frameworks required. Just clone the repository and open any `.html` file in your browser.

```bash
git clone https://github.com/PedroHFelix-hub/studies-about-internet.git
cd studies-about-internet
```

Then open `index.html` in your preferred browser.
